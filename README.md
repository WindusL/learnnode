**MarkDown简单语法**

1. \# 一级标题

\## 二级标题

\### 三级标题
依此类推，总共六级

2. 列表

文字前加上 - 或 * 即可变为无序列表

有序列表则直接在文字前加1. 2. 3. 符号要和文字之间加上一个字符的空格

3.引用　文本前加入 > 
>测试引用

4. 图片与链接
  图片为：\!\[](){ImgCap}{/ImgCap}
  
  链接为：\[]()
  
  插入图片的地址需要图床，这里推荐围脖图床修复计划 与 CloudApp 的服务，生成URL地址即可。

5. 粗体与斜体
   用两个 * 包含一段文本就是粗体的语法，用一个 * 包含一段文本就是斜体的语法。
   
6. 分割线　三个 */_/- 号
